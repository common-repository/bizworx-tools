== Description ==
 
This plugin is used to register some of custom fileds for the Bizworx theme. It is meant to be only used with Bizworx theme.
 
Few notes:
 
*   Registers custom fields and posts
*   Meant to be used with Bizworx wordpress theme
 
== Installation ==
 
1. Go to Plugins > Add New and search for Bizworx Tools.
2. Once you find it just install and active.
